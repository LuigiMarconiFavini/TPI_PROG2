# Frutales del Norte

Aplicación web desarrollada en Python utilizando el framework Flask como parte del Trabajo Práctico Integrador de Programación 2.

El proyecto implementa un servidor web con manejo de rutas, renderizado de templates y gestión de base de datos.


## 🧠 Tecnologías
- Python
- Flask 
- HTML5
- CSS
- JavaScript

## 🚀 Instalación
1. Clonar el repositorio
   
   `git clone https://github.com/LuigiMarconiFavini/TPI_PROG2.git`
   
3. Entrar a la carpeta
   
   `cd TPI_PROG2`
   
5. Crear y activar entorno virtual
   
   `python3 -m venv venv`
   
   `export FLASK_APP=app.py`
   
   `source venv/bin/activate`
   
   `flask init-db (solo una vez)`
   
7. Ejecutar el proyecto
   
   `para ejecutar el proyecto es: Python -m flask run`
   
9. Instalar dependencias
    
   `pip install -r requirements.txt`
